## 📌 프로젝트 개요
Vue + TypeScript + Vite를 활용하여 제작한 한가위 테마의 정적 웹페이지입니다.  
달빛 아래 토끼가 떡방아를 찧는 한가위의 따뜻한 분위기를 표현했습니다.
