# 🌕 한가위 웹페이지 (Chuseok Festival Page)

> [🚀 배포된 페이지 바로가기](https://ehyewon.github.io/vue-static-page/)

## 📌 프로젝트 개요
Vue + TypeScript + Vite를 활용하여 제작한 한가위 테마의 정적 웹페이지입니다.  
달빛 아래 토끼가 떡방아를 찧는 한가위의 따뜻한 분위기를 표현했습니다.

---

## 💻 실행 방법

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build

# GitHub Pages 배포 (gh-pages 사용)
npm run deploy
